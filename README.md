# ・・・・・・・・・第8回定期公演「Tokyo in Words and Letters」第2部「Letters」の来場者特典でもらった『・ちゃんからのお手紙』（モールス信号が録音されている音声ファイル）を解読するためのデコーダー

- 長い／(^o^)＼

## System Requirements

- Node.js

## Install

``` bash
$ npm install
```

## Usage

``` bash
$ node . < ・ちゃんからのお手紙.wav
```
